# Multimodal Talent Intelligence System

## 🚀 Overview
AI-based recruitment system that evaluates candidates beyond keyword matching.

## 🔍 Key Features
- Multi-source data analysis (resume, projects)
- Context-aware scoring system
- Explainable evaluation logic
- Reduces bias in hiring

## 🛠 Tech Stack
Python | NLP | ML | Backend

## ⚙️ Architecture
Input Data → Feature Extraction → Scoring Engine → Output Ranking

## 📊 Impact
Improves candidate evaluation accuracy vs traditional ATS
